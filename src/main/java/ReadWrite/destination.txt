В основе всей системы ввода-вывода Java лежат два абстрактных класса:

InputStream - это абстрактный суперкласс для всех классов, представляющих входной поток байтов. Его ключевые методы: read(), read(byte[]), close().

OutputStream - это абстрактный суперкласс для всех классов, представляющих выходной поток байтов. Его ключевые методы: write(int), write(byte[]), flush(), close().
